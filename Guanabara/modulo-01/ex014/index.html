Estilos CSS internos

No código HTML dessa página vemos um exemplo de CSS local, que é mais vantajosa que a inline, usada bastante no dia-a-dia

Essa técnica possui duas desvantagens:

  • Geralmente o estilo ocupa muito espaço no código, jogando o HTML lá pra baixo
  • Ela funciona apenas na página selecionada (no atual caso apenas no index.html
