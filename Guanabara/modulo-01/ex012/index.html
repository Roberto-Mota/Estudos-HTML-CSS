Estilo inline

Aqui eu estava utilizando (comentei para nao aparecer mais) um método que já nao está em conformidade, precisamos adotar o CSS

Para converter essa má utilização de método, pode-se adotar um estilo para o próprio body através de uma declaração de CSS

Pontos positivos:

  • Pode haver uma marcação específica para cada Tag, e.g. cada h1 ou p ter sua própia cor
  • Haver pequenas variações entre cada tag
Pontos negativos:
  • Pode criar uma quantidade grande de códigos repetidos
  • Pode dar mais trabalho de criar e manter do que da outra forma
