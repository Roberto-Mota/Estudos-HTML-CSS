Usando o id com CSS

Tudo que
em HTML é id => em CSS é # (1 elemento)
em HTML é class => em CSS é . (n elementos)

Dentro de um mesmo documento HTML só pode ter um elemento com um id, isto é, dá pra usar um mesmo id em várias tags do HTML mas isso é um comportamento errado, afinal funcionar não significa que está certo. Para evitar esse comportamento podemos utilizar classes

Para poder utilizar atributos globais existe a tag <span>, que não significa nada por si só mas pode carregar alguma classe ou id nela

Também é possível acumular classes e id em um mesmo elemento. mas precisa levar em consideração a hierarquia das aplicações do CSS, pois o Class é aplicado antes e o ID depois, sobrepondo as configurações do ID sempre herdando a classe, que atrás de tudo herda as configurações da própria tag, ou seja: Tag < Class < ID

Hierarquia

Essa hierarquia de ID, classes e tag funcionam de forma semelhante à hierarquia de inline, local e external, onde, por peso, se configura assim:

  • 1° (mais pesado): inline / ID
  • 2° (intermediário): local / Class
  • 3° (mais leve): external / TAG
