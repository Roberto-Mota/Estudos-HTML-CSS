Fazendo um degradê

body {
        background-image: linear-gradient(to right, white, blue);
    }

Esse formato faz um degradê simples, da esquerda pra direita, do branco para o azul

Para não ficar com um degradê repetindo no fundo enquanto da um scroll down precisamos utilizar o atributo(?) background-attachment: fixed;. Com esse atributo, o gradiente fica fixado na tela como um todo (meio feio eu acho)
