Modelo de Caixas

Atributos de uma caixa

  • Height: Altura
  • Width: Largura
  • Border: Linha que circunda o conteúdo
  • Padding: (Tradução: preenchimento) Espaço dado entre o border e o pontilhado da caixa (?)
  • Margin: Espaço externo da borda entre dois boxs
  • Outline: Espaço fora da borda dentro do Margin

Tipos de boxs

  • box-level: ao ser criada, pula para uma linha nova, ou seja, para um próximo espaço, ocupando uma linha inteira do viewport
    Exemplos: <div>,<h1>,<p>,<main>,<nav>,<article>,<header>,<aside>,<footer>,<form>,<video>, etc.
  • inline-level: ao ser criada continua o caminho do conteúdo, ou seja, não quebra a linha e nem ocupa todo o width do viewport.
    Exemplos: <span>,<a>,<code>,<strong>,<small>,<sup>,<sub>,<em>,<label>,<button>,<input>,<select>

    • Anotações

    • User agent: O navegador que está abrindo o conteúdo

    Exemplos

    Títulos são exemplos de box-level

    Também é possível usar short-hand

    Parágrafos também são exemplos de box-level, porém os links são exemplos de inline-level. Mudar o display significa mudar se a tag será inline-level ou box-level
